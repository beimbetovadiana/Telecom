# PRD: KPI Dashboard для управления продажами подразделений

Версия: 1.0  
Автор: Диана Беймбетова

## 1. Цель проекта

Разработать веб-приложение для автоматизации управления плановыми показателями подразделений.

Система должна заменить ежедневную работу с Excel-файлами, автоматически рассчитывать показатели продаж и отображать результаты в интерактивном Dashboard.

## 2. Пользователи

### Администратор

Функции:

- создание пользователей;
- управление подразделениями;
- загрузка планов;
- загрузка ежедневных файлов продаж;
- просмотр Dashboard;
- просмотр аналитики;
- экспорт отчетов;
- изменение настроек системы.

### Пользователь

Функции:

- вход в систему;
- просмотр планов;
- внесение фактических показателей;
- просмотр собственного результата.

## 3. Роли

- Administrator
- Manager
- Employee

## 4. Основной функционал

### Импорт планов

Администратор загружает Excel.

План содержит показатели:

- Продажи в деньгах
- Отток
- ШПД
- ТВ
- FMS
- другие KPI

Планы могут загружаться:

- на день
- на неделю
- на месяц

После импорта система автоматически распределяет планы по подразделениям и сотрудникам.

## 5. Импорт ежедневного файла продаж

Администратор ежедневно загружает Excel.

Каждая строка файла содержит информацию по лицевому счету:

- Лицевой счет
- Подразделение
- Сотрудник
- Услуга
- Сумма
- Дата

После загрузки система автоматически:

- определяет продажи;
- определяет отток;
- суммирует показатели;
- обновляет Dashboard.

## 6. Правила обработки

### Продажи

Если сумма > 0, сумма относится к показателю `Продажи`.

### Отток

Если сумма < 0, сумма относится к показателю `Отток`.

### ШПД

Показатель объединяет:

- FTTX
- ADSL
- БШПД
- GPON

Список настраивается.

### ТВ

Все услуги телевидения.

### FMS

Все мобильные услуги.

## 7. Dashboard

Dashboard является главным экраном системы.

Структура:

```text
ИТОГО ПО ОБЛАСТИ
↓
Подразделение
↓
Сотрудник
```

Для каждого показателя отображается:

- План
- Факт
- %

Пример:

| Подразделение | Продажи План | Продажи Факт | Продажи % | Отток План | Отток Факт | Отток % | ШПД План | ШПД Факт | ШПД % | ТВ План | ТВ Факт | ТВ % | FMS План | FMS Факт | FMS % |
|---|---:|---:|---:|---:|---:|---:|---:|---:|---:|---:|---:|---:|---:|---:|---:|
| ИТОГО ПО ОБЛАСТИ |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |

Первая строка `ИТОГО ПО ОБЛАСТИ` закрепляется сверху.

Каждое подразделение раскрывается:

```text
▼ Караганда
  Иванов
  Петров
  Сидоров
```

## 8. Настройки Dashboard

Администратор может включать или выключать отображение:

- План
- Факт
- %
- Только %
- Только факт
- Только план

## 9. Автоматический расчет

После каждой загрузки файлов система автоматически рассчитывает:

- Продажи
- Отток
- ШПД
- ТВ
- FMS
- %
- Итоги подразделений
- Итоги области

## 10. Напоминания

- Суббота, 17:00: проверка.
- Воскресенье: уведомления не отправляются.
