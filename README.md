# Telecom KPI Dashboard

Статический сайт на основе PRD `KPI Dashboard для управления продажами подразделений`.

## Что входит

- главный экран Dashboard;
- KPI-карточки: продажи, отток, ШПД, ТВ, FMS, подразделения, сотрудники и неотчитавшиеся;
- таблица с иерархией `ИТОГО ПО ОБЛАСТИ → подразделение → сотрудник`;
- переключение отображения `План / Факт / %`;
- поиск по подразделению, сотруднику и KPI;
- блоки импорта Excel, отчетов, настроек KPI и истории изменений.

## Как открыть

Откройте файл `index.html` в браузере.

## Как разместить на GitHub Pages

1. Загрузите файлы `index.html`, `styles.css`, `script.js`, `README.md` и `PRD_KPI_Dashboard.md` в репозиторий `beimbetovadiana/Telecom`.
2. В GitHub откройте `Settings → Pages`.
3. В разделе `Build and deployment` выберите `Deploy from a branch`.
4. Укажите ветку `main` и папку `/root`.
5. Сохраните настройки.

После публикации сайт будет доступен по адресу:

`https://beimbetovadiana.github.io/Telecom/`
